### Chatbot ###

##  About

This is a Python-based chatbot that can answer questions using:

* Rule-based logic
* Machine Learning model


##  Features
* 💬 Chat-based interaction
* 🧠 Machine Learning prediction
* 📝 Chat history tracking
* 🔤 Spell correction


##  Technologies Used
* Python
* Scikit-learn
* TextBlob
  

##  How to Run
 1. Train the model
python train.py
 2. Run chatbot
python chatbot.py


##  Project Structure

* chatbot.py → Main chatbot program
* train.py → Train ML model
* intents.json → Dataset for chatbot

##  Note
* Run `train.py` before running chatbot
* `.pkl` files are generated after training
